# Certificates

## Get all certificates

```php
$unit->getCertificates();
```

## Get certificate

```php
$unit->getCertificate('example');
```

## Upload certificate

```php
$unit->uploadCertificate('/var/www/certififcates/example.pem','example');
```

## Remove certificate

```php
$unit->removeCertificate('example');
```
